# 6. Platform Architecture & Tech Stack

| Layer        | Technology                         | Purpose                      |
| ------------ | ---------------------------------- | ---------------------------- |
| Web Frontend | SvelteKit + Svelte 5 + Tailwind v4 | Premium UI, SSR, SEO         |
| Mobile       | Progressive Web App                | Offline resilience           |
| API          | FastAPI + Go Fiber                 | High concurrency             |
| Real-Time    | Bun + Elysia + SSE                 | Live updates                 |
| Database     | PostgreSQL                         | Multi-tenant relational core |
| Cache        | Redis                              | State and performance        |
| Async        | Celery + Go Workers                | Background processing        |
| Storage      | S3-Compatible Storage              | Files and documents          |
| Payments     | Moyasar                            | Regional payment processing  |
| Messaging    | Unifonic + Amazon SES              | Communication delivery       |
| Hosting      | AWS Riyadh / GCC Cloud             | Data residency               |

## Architectural Principles

* API-first
* Tenant isolation
* Row-level security
* Explicit state machines
* Immutable ledgers
* Idempotency everywhere
* Horizontal scaling
